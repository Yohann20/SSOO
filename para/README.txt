Para compilacion y ejecucion del programa secuencial es necesario lo siguiente:
- Librerias/Dependencias actualizadas de OpenCv  

- Para compilar paralelo.cpp se necesita el siguiente comando en la terminal: " make " 
- Para ejecutar paralelo.cpp se necesita introducir el siguiente comando en la terminal: 
" make run_threads imagen3.jpg imagen3_en_gris.jpg NUMERO_DE_HILOS " Teniendo en cuenta que \
NUMERO_DE_HILOS es un numero natural cualquiera que determina el numero de hebras que se quiere
utilizar para ejecutar el programa.
